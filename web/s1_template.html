PC remote control panel

Connecting guide

1- Install the PCRemoteControl App! You can find the apk along with the source code on github
2- Find the correct IP There may be many IP addresses assigned to your PC device. Enter the correct address in the app. We have found the ones below.
    %IPS%
You can find a similar list on your mobile app. You need to match the beginning or the "subnet" of an ip address on your phone to one on the computer for example: 192.168.0.103 192.168.0.56 is a match. Once you find a match enter the PC ip address in the "address" field on your phone.
